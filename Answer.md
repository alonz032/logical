One challenge during this activity was understanding the difference between the XOR and TEST instructions. Although both perform logical operations, XOR stores its result while TEST only updates the processor flags. Another challenge was learning how the Zero Flag controls conditional jumps such as JZ. Debugging the program with GDB helped verify that xor eax, eax correctly cleared the register and that test ebx, 1 correctly identified whether the number was even or odd. After stepping through the instructions and observing the register values, the program worked as expected.
